# Gestión de Inventario ABB

## Introducción

Este es un proyecto de gestión de inventario que utiliza un **Árbol Binario de Búsqueda (ABB)** como estructura de datos principal. El ABB permite organizar los productos de manera jerárquica, facilitando búsquedas rápidas y eficientes.

La aplicación permite gestionar un inventario de productos, realizando operaciones como inserción, búsqueda y eliminación de items de forma optimizada.

## Archivos

- `Main.java` - Punto de entrada del programa
- `InventarioController.java` - Controlador del inventario
- `ArbolBinarioBusqueda.java` - Implementación del árbol binario
- `NodoBinario.java` - Nodo del árbol
- `IArbolBinario.java` - Interfaz del árbol
